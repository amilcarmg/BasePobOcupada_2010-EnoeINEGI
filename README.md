# BasePobOcupada_2010-EnoeINEGI
# ENOE 2010 para Econometría Espacial: Base de datos para docencia e investigación.
Esta base de datos se construye a partir de la Encuesta Nacional de Ocupación y Empleo 2010 (ENOE), levantada por el Instituto Nacional de Estadística y Geografía (INEGI). 

Dicha encuesta constituye una fuente fundamental de información sobre el mercado laboral en las zonas urbanas de México. Los datos aquí contenidos corresponden exclusivamente a la población ocupada e incluyen, para cada uno de los municipios encuestados durante 2010, el factor de expansión correspondiente, el cual garantiza la representatividad de los resultados a nivel urbano. Asimismo, se incorporan variables sociodemográficas y laborales como sexo, edad, años de escolaridad e ingreso por hora percibido por los trabajadores en su ocupación principal.
Fuente: Encuesta Nacional de Ocupación y Empleo 2010 (ENOE-INEGI).

## Descripción
Este repositorio contiene una base de datos en formato **Stata (.dta)** elaborada para apoyar la enseñanza de la asignatura bEconometría Espacial en la maestría en Planeación Espacial del CentroGeo.
La base fue construida mediante la selección, integración y armonización de variables provenientes de la **Encuesta Nacional de Ocupación y Empleo 2010 (ENOE)** del Instituto Nacional de Estadística y Geografía (INEGI), con el propósito de facilitar ejercicios de análisis econométrico espacial.
